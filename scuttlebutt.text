---
title: Scuttlebutt
---

A goofy name for a simple replication algorithm described in [*Efficient Reconciliation and Flow Control for Anti-Entropy Protocols*](https://www.cs.cornell.edu/home/rvr/papers/flowgossip.pdf).

## Implementations

- Java
    - [Cassandra](https://wiki.apache.org/cassandra/ArchitectureGossip)
- JavaScript
    - [hyperlog](https://github.com/mafintosh/hyperlog) ([Bliki](hyperlog))
    - [Scuttlebot](https://github.com/ssbc/scuttlebot) ([Bliki](scuttlebot))
    - [secure-scuttlebutt](https://github.com/ssbc/secure-scuttlebutt)
    - [simple-scuttle](https://awinterman.github.io/simple-scuttle/)
