replikativ

A Clojure library for P2P replication with CRDTs. Source code available on GitHub.

Getting Started

Features

  • Provides a CDVCS data type which represents an append-only graph (similar to hyperlog)
  • Expects the developer to be aware of possible conflicts
  • Presents an API to allow those conflicts to be resolved explicitly

Example

See replikativ/replikativ-cljs-demo on GitHub.
